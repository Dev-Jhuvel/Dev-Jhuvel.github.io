Ideas: Twitter Clone Website

The Idea Project is a clone of the website Twitter/X made with HTML, CSS, PHP, JavaScript, PHP and Laravel framework. Any user can create an account and login. The login user can post Idea in the newsfeed and comment in own post or other post, the login user can like any post in the feed. Login user can follow and unfollow any user, it can also modify the user profile. This project aims to practice the basics in Laravel, the use of Model, Route, Controller and View with Bootstrap also the CRUD functionality. The project is deployed in the web using the free hosting with cPanel and can be access.

Home Brew: Content Management System

The Home Brew Project is made with HTML, CSS, PHP, JavaScript, Vanila PHP and Bootsrap. Only Admin user can create an account and login.
